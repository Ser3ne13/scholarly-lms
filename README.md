# Scholarly LMS

A comprehensive School Learning Management System built with pure PHP and MySQL.

## Features

### 👑 Admin Role
- Dashboard with real statistics
- User management (Create, Edit, Delete)
- Subject management with school level organization
- Section management (Kindergarten to Grade 12)
- Multiple view modes (Grid, Compact, List)
- Master schedule view

### 👨‍🏫 Teacher Role
- Subject management
- Lesson creation
- Student management
- Grade tracking

### 🎓 Student Role
- View enrolled subjects
- Access lessons
- Track progress
- Submit assignments

## Technology Stack
- PHP 8.x (Pure PHP, no frameworks)
- MySQL
- HTML5/CSS3
- JavaScript (vanilla)
- Modern responsive design

## Installation

1. Clone this repository
2. Import `database.sql` to MySQL
3. Configure `config/database.php`
4. Run on XAMPP/WAMP/LAMP
